<h1>Kimetsu Memory</h1>
1. Visão geral

Jogo da memória (concentração) que exibe uma grade de cartas viradas para baixo. O jogador clica em duas cartas por vez para revelar imagens — se as imagens coincidirem, elas permanecem viradas para cima; caso contrário, viram de volta. O objetivo é encontrar todos os pares com o menor número de movimentos e no menor tempo possível.

Público-alvo: fãs do anime, iniciantes em front-end que querem um projeto prático.

2. Recursos (features)

Layout responsivo (mobile → desktop).

Mecânica clássica de virar cartas (flip) e verificar pares.

Contador de movimentos e cronômetro.

Indicador de progresso / pares encontrados.

Animações suaves (CSS) ao virar cartas e ao completar o jogo.

Modal de fim de jogo com estatísticas (tempo, movimentos) e botão para reiniciar.

Opções simples para configurar dificuldade (quantidade de pares).

Acessibilidade básica: foco por teclado, descrições alt nas imagens.

3. Tecnologias

HTML5

CSS3 (Flex / Grid e transições)

JavaScript (ES6)
